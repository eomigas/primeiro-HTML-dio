Oque é HTML?

(Voltar)

O HTML(Linguagem de Marcação de Hipertexto) nada mais é que uma linguagem de marcação usada no desenvolvimento de páginas web ela consiste em uma série de elementos que você usa para delimitar diferentes partes do conteúdo e o modo como elas aparecem As tags anexas podem transformar uma palavra ou imagem num hiperlink, pode colocar palavras em itálico, pode aumentar ou diminuir a fonte e assim por diante.

Elementos

(Voltar)

Um elemento é parte da pagina web Em HTML, um elemento pode conter um item de dados, um bloco de texto, uma imagem ou talvez nada. Um elemento típico inclui uma tag de abertura com alguns atributos, o conteúdo de texto incluído e uma tag de fechamento.

Tags

(Voltar)

A as chamadas TAGS em HTML são usadas para criar elementos
